# MoveTrack

MoveTrack es un dashboard de entrenamiento hecho con Astro.

Permite:
- sumar km corridos
- agregar entrenamientos
- modificar objetivos desde la página

## Tecnologías usadas

- Astro
- HTML
- CSS
- JavaScript

## Cómo ejecutar el proyecto

1. Instalar dependencias:
npm install

2. Ejecutar el servidor:
npm run dev

3. Abrir en el navegador:
http://localhost:4321
